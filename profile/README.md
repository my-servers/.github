# MyServers

![](../img/app_store.jpg)

一个App监控管理你所有的服务器和服务端应用。这是一款监控管理App，可以扩展各种服务端插件，十分灵活。


## 特性
- 监控管理服务器，应用
- 插件化，可扩展性十分强大
- 持续支持各种插件的开发
- 所有数据加密传输，保证安全
- 丰富的UI
